# Website Nội Thất Hoàng Hoan - Frontend UI/UX
## Tác Giả : Lê Đình Hùng -- .NET Developer

### 📝 Giới thiệu dự án
Dự án tập trung xây dựng giao diện người dùng chuyên nghiệp cho đơn vị cung cấp nội thất HOÀNGHOAN . Mục tiêu chính là cung cấp trải nghiệm xem sản phẩm trưng bày trực quan, tra cứu thông tin dịch vụ dễ dàng và tối ưu hóa tương tác người dùng.

###  Vấn đề giải quyết
* UI/UX : Cải thiện trải nghiệm người dùng thông qua giao diện hiện đại, dễ điều hướng.
* Responsive : Hiển thị tối ưu trên đa thiết bị (Mobile, Tablet, Desktop) giúp tiếp cận người dùng mọi lúc mọi nơi.
* Data Process : Xử lý render dữ liệu động linh hoạt thay vì fix cứng HTML.

###  Các tính năng đã hoàn thiện

* Layout : Hoàn thành Code Header/Footer và các trang con (Trang chủ, Sản phẩm, Tin tức, Liên hệ).
* Phù hợp nhiều thiết bị : Hoàn thành Responsive mượt mà cho đa dạng chế độ xem ( Desktop , Tablet và Mobile)
* Render dữ liệu : Sử dụng JS + DOM/jQuery để hiển thị danh sách sản phẩm và bài viết.
* Thành phần bổ trợ : Tích hợp Slick Slider cho phần trình chiếu banner và đối tác.

### Công nghệ sử dụng
* Frontend Core: HTML5, CSS3 .
* Libraries: JavaScript , jQuery, Bootstrap , Slick
* Tools: Visual Studio Code, Git , Github
